# 世界杯体彩策略顾问系统 - Demo

基于多模型融合（泊松回归 + Elo动态评分 + Clement经济模型 + 状态模型）的世界杯竞彩策略分析工具。

## 核心特性

- **多模型融合预测**：泊松回归、Elo动态评分、Clement经济模型、近期状态模型四模型加权融合
- **价值投注识别**：Value = P × Odds - 1，仅推荐 Value > 8% 的机会
- **凯利公式仓位**：1/4 Kelly 分额，单场上限 5%
- **狂热系数对冲**：综合社媒热度、名人共识、魔咒周期，反向衰减过度热门方向
- **人机协同闭环**：系统出策略 → 人工决策 → 赛后反馈 → 模型进化

## 技术栈

- Python 3.8+ / Flask
- NumPy / SciPy（泊松分布计算）
- BeautifulSoup4 / Requests（数据采集）
- ECharts / Mermaid（可视化）
